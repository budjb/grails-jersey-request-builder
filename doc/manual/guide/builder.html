2 Making Requests

The RequestBuilder can be built and used in several ways, depending on how the author intends to use it.

2.1 HTTP Methods

The HTTP specification defines several request methods. Each method has a different intended use, and there are some behavior requirements around them.

The RequestBuilder supports many of these methods, as outlined below:

MethodSupportedRequestBuilder Method
GETSupportedget()
POSTSupportedpost()
PUTSupportedput()
DELETESupporteddelete()
HEAD Unsupported  
OPTIONS Unsupported  

The unsupported methods listed above will be added in the near future.

2.2 Simple Usage

The simplest way to use the RequestBuilder is to use the convenience methods that only require a URL, and optionally the body on methods that support it. They can be used as follows: 
// GET example
new RequestBuilder().get("http://www.example.com")
// POST example
new RequestBuilder().post("http://www.example.com", "payload")
// PUT example
new RequestBuilder().put("http://www.example.com", "payload")
// DELETE example
new RequestBuilder().delete("http://www.example.com")

2.3 Closure Usage

Using the closure-style functions is the most common use of the RequestBuilder. These methods provide an easy to use, centralized way to build a request with varying parameters. This documentation will not go into detail about all of the options available with the RequestBuilder, but all of these options are documented completed in the reference section.

All of the HTTP methods can use the closure-based methods.

new RequestBuilder().post {
    uri = "http://www.example.com"
    body = ["foo": "bar"]
    headers = [
        "X-Auth-Key": "1321d980-3ceb-11e3-b872-081196ac8a64"
    ]
}

2.4 Stateful Usage

The RequestBuilder also allows the author to build a request gradually. All of the properties that can be set inside of the closure-based methods can also be set directly on the object itself.

RequestBuilder builder = new RequestBuilder()
builder.uri = "http://www.example.com"
builder.accept = "text/html"
def result = builder.get()
