Provides the Jersey client library and a builder to simplify use of the library.

(Quick Reference)

skipStatusCheck

Purpose

Whether to prevent exceptions for non-successful HTTP status codes.

Examples

jerseyRequestBuilder.get {
    uri = "http://example.com"
    skipStatusCheck = true
}

Description

If true, prevent the Jersey Request Builder from throwing an exception if a non-successful HTTP status code is encountered.
